# web-socket

## 課題

簡単なWebSocketサーバとクライアントを書いてください。(echoサーバやチャットなど…)
当日使う環境でnodejs,wsを使っていますが、言語やライブラリは特に指定しません。

## やったこと
* npmを使用して必要モジュールをインストール
	* `npm install ws express`
* `app.js`と`index.html`を作成し、ソケット通信を実行
	* `node app.js`
	* `http://localhost:3000/index.html`
